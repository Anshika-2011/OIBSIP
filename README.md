# Customer Segmentation using Machine Learning

## Project Overview
This project performs Customer Segmentation using the Online Retail dataset. Customers are grouped into different clusters based on their purchasing behavior using RFM (Recency, Frequency, Monetary) analysis and K-Means Clustering.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Steps
1. Import Dataset
2. Data Cleaning
3. RFM Analysis
4. Feature Scaling
5. K-Means Clustering
6. Cluster Analysis
7. Customer Segmentation

## Files
- CUSTOMER SEGMENTATION.ipynb
- Online Retail Customer Segmentation.csv
- README.md

## Result
Customers are divided into different segments based on purchasing behavior, helping businesses target customers more effectively.
